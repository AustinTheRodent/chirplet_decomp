main args:
    -all
        gen sim setup values, run SW sim, run HW sim
    -hw
        gen sim setup values, run HW sim
    -sw
        gen sim setup values, run SW sim
    -cmp
        compare HW and SW outputs
    -clean
        remove generated files

optional args:
    -i <user input file name>
        use user input
    -n <number of samples>
        set number of samples
